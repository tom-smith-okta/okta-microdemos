This demo shows SSO from Okta to AWS.
The Okta app is a "vanilla" SAML app that calls a SAML inline hook.
The SAML inline hook sends the user info (including group membership) to an external service.
The external service looks at the user's group memberships and sends back to Okta a patch with the AWS accounts and roles to be included in the SAML assertion.

What to do
Log in as user:
diana.prince
Then, click on the link for "AWS - raw SAML with inline hook". Diana will arrive at the AWS console.
