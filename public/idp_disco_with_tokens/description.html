This demo shows an example of IDP discovery using the Okta widget, and also federation via OIDC to another Okta tenant.
After authenticating, the user will have a session in both tenants, and will have tokens in local storage as well.
What to do
  • Launch the widget.
  • enter username nick.fury@idpx.com
  • you will be redirected to idpx.oktapreview.com
  • enter the password for nick.fury@idpx.com
  • you will be redirected back to this page
  • you will have a session with both Okta tenants
  • you will get an id token and an access token
